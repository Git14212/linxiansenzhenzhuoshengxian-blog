# 飞狐生鲜策划博客

## 🎯 项目概述

这是一个为生鲜策划从业者和学习者创建的专业博客网站。由飞狐（10年+生鲜策划导师）创建和维护，旨在分享实战经验、分析最新案例、收藏策划方案。

## 📁 文件结构

```
生鲜策划博客/
├── index.html           # 首页
├── blog.html           # 策划心得博客页
├── cases.html          # 案例展示页
├── collection.html     # 自媒体方案收藏页
├── resources.html      # 资源下载页
├── about.html          # 关于我页面
├── style.css           # 统一样式表
├── README.md           # 项目说明（本文件）
└── 关于博客建设.md     # 详细建设文档
```

## 🚀 快速开始

### 本地预览
1. 双击 `index.html` 即可在浏览器中打开首页
2. 或右键选择其他浏览器打开
3. 所有页面均已链接，可直接在页面间导航

### 在线部署
推荐部署到以下平台（任选其一）：

#### GitHub Pages（最推荐）
```bash
# 1. 创建GitHub仓库
# 2. 将整个文件夹上传
# 3. 进入仓库设置 → Pages
# 4. 选择main分支 → 保存
# 5. 访问 https://你的用户名.github.io/仓库名
```

#### Netlify
1. 访问 https://app.netlify.com
2. 拖拽整个文件夹上传
3. 自动部署，获得 https://xxx.netlify.app 网址

#### Vercel
1. 访问 https://vercel.com
2. 导入GitHub仓库
3. 自动部署，获得 https://xxx.vercel.app 网址

## 📝 内容更新指南

### 添加新博客文章
1. 打开 `blog.html`
2. 在 `<!-- 博客文章列表 -->` 区域添加新的文章HTML
3. 格式参考现有文章结构
4. 更新文章计数和最新文章展示

### 添加新案例
1. 打开 `cases.html`
2. 在对应的分类区域添加新的案例卡片
3. 格式参考现有案例结构

### 更新收藏方案
1. 打开 `collection.html`
2. 在对应平台区域添加新的收藏项
3. 保持日期、链接、描述信息完整

### 添加新资源
1. 打开 `resources.html`
2. 在资源列表区域添加新的资源项
3. 提供清晰的下载链接和描述
